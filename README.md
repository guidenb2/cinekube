# cinekube

This is the backend component of the CineKube microservice application

## Installation

Install the project dependencies
    
    $ lein deps

Install the project

    $ lein install

Generate the project uberjar

    $ lein uberjar

## Usage

Run the jar file

    $ java -jar cinekube-0.1.0-standalone.jar [args]
